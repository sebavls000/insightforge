# InsightForge

Local-first AI analyst for real business files.

Upload CSV, Excel, and PDF files to get:
- automatic insights
- anomaly detection
- charts
- data quality checks
- executive summaries
- exportable reports

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- FastAPI
- Python
- Pandas
- scikit-learn
